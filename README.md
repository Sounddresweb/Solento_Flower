Solento Flower - Prototipo Interactivo 2D
Solento Flower es un videojuego con perspectiva top-down desarrollado en Unity como parte del proceso formativo en Ingeniería Multimedia de la UNAD. El proyecto pone al jugador en el rol de una ingeniera ambiental encargada de restaurar el equilibrio en un ecosistema afectado, con el objetivo final de recuperar el "Gran Sol".

🛠️ Características Técnicas
Motor de Juego: Unity 2022.3 (LTS).

Perspectiva: Top-Down 2D.

Sistema de Audio: Implementación de Audio Mixer con canales separados para SFX y Música, utilizando estándares de sonoridad para WebGL.

Optimización: Sistema de Object Pooling para la gestión eficiente de proyectiles.

IA de Enemigos: Sistema de seguimiento y ataque basado en áreas de activación (Triggers).

🕹️ Controles
Movimiento: Teclas W, A, S, D.

Disparo: Clic Izquierdo del mouse.

Interacción: El combate se activa automáticamente al entrar en zonas de conflicto.
